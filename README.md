# Lumen ARIA Blocks

Lumen ARIA Blocks is a WordPress plugin that adds accessibility-focused Gutenberg blocks.

It is designed for editorial teams that need interactive UI components with solid keyboard and assistive-technology behavior by default.

## Highlights

- Keyboard-first interaction patterns.
- SSR-first output for resilient rendering and content continuity.
- Progressive enhancement (content is still present server-side).
- Theme-agnostic behavior (works with standard WordPress themes).
- Privacy-friendly runtime (no telemetry, no tracking scripts).

## Included Blocks

- Accordion
- Button (action/link/toggle)
- Carousel
- Dialog
- Popup
- Tabs
- Tooltip

## Requirements

- WordPress 6.5+
- PHP 7.4+

## Installation (From Source)

1. Copy this repository folder to `wp-content/plugins/lumen-aria-blocks`.
2. Activate **Lumen ARIA Blocks** from the WordPress Plugins screen.
3. In the block editor, use the **Lumen Accessibility** category.
